🚀 Запуск на Termux

Этот файл уже готов к работе с сервером. Следуйте инструкции ниже.

1Установить зависимости в Termux

Установи Termux из F-Droid (не Google Play), затем выполни:

pkg update && pkg upgrade -y pkg install python -y pip install fastapi uvicorn aiofiles python-multipart

2Создать папку и server.py

mkdir ~/mythoria && cd ~/mythoria nano server.py

Вставь содержимое server.py (см. следующий шаг), сохрани Ctrl+X → Y → Enter

3Содержимое server.py

from fastapi import FastAPI from fastapi.middleware.cors import CORSMiddleware from fastapi.staticfiles import StaticFiles from pydantic import BaseModel import json, os, time app = FastAPI() app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_methods=["*"], allow_headers=["*"]) DB = "posts.json" class Post(BaseModel): title: str; content: str; type: str = "fanfic" fandom: str = ""; author: str = "Аноним"; rating: int = 4 def load(): if not os.path.exists(DB): return [] return json.load(open(DB, encoding="utf-8")) def save(d): json.dump(d, open(DB,"w",encoding="utf-8"), ensure_ascii=False, indent=2) @app.get("/posts") def get_posts(): return load() @app.post("/posts") def add_post(p: Post): posts = load() new = {**p.dict(), "id":"srv-"+str(int(time.time())), "date":"Только что","views":0,"replies":0,"likes":0,"hot":False} posts.insert(0, new); save(posts); return new app.mount("/", StaticFiles(directory=".", html=True), name="static")

4Скопировать index.html

cp /sdcard/Download/index.html ~/mythoria/index.html

Или перенеси файл через Telegram → сохрани в загрузки → скопируй командой выше.

5Запустить сервер

cd ~/mythoria uvicorn server:app --host 0.0.0.0 --port 8000

Сайт доступен локально: http://localhost:8000

6Открыть доступ из интернета (ngrok)

В новой вкладке Termux (свайп слева → New Session):

pkg install wget -y wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-arm64.tgz tar xf ngrok-v3-stable-linux-arm64.tgz && chmod +x ngrok ./ngrok config add-authtoken ВАШ_ТОКЕН # с сайта ngrok.com ./ngrok http 8000

Ngrok покажет ссылку вида https://abc123.ngrok-free.app — отправляй её друзьям!

7Статус подключения

Индикатор в левом нижнем углу показывает: ● Сервер — посты сохраняются для всех, ● Локально — работает только на вашем устройстве.
